# KIDDYLearn (Technovation Girls Quarterfinalist) 🏆

**An educational mobile application designed to empower children by teaching essential life skills through gamified interactive lessons.**

## ✨ Project Overview
**KIDDYLearn** (developed under the team name *Lil-Meerkats*) is a mobile solution created for the **Technovation Girls** international competition. The app provides a safe, engaging environment where children can learn about gardening, cooking, street safety, and household chores.

### 🎖️ Recognition
* **Quarterfinalist** at Technovation Girls (International Mobile App Competition).

## 🚀 Key Features
* **Gamified Learning:** Interactive modules covering diverse topics:
  * 🚦 **Street Safety:** Learning traffic signs and safe crossing procedures.
  * 🍳 **Cooking Basics:** Step-by-step simple recipes like muffins and omelettes.
  * 🌱 **Gardening:** Instructions on planting and nurturing nature.
  * 🧹 **Household Skills:** Making chores like sweeping and floor cleaning fun.
* **Cross-Platform Experience:** Developed using **Apache Cordova** for mobile accessibility.
* **Real-time Progress:** Integrated with **Firebase** for secure user authentication and data storage.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Mobile Framework:** Apache Cordova
* **Backend (BaaS):** Firebase (Authentication & Cloud Firestore)
* **Design & Prototyping:** Figma

## 📊 Beyond the Code
This project involved a complete startup lifecycle:
* **Business Plan:** Conducted market analysis, branding, and financial projections.
* **Pitch Video:** Created a presentation for international judges explaining the social impact.
* **UI/UX Design:** Extensive prototyping in Figma to ensure a kid-friendly interface.
